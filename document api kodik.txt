API предназначено для удобного получения любой информации из нашей базы.
Использовать API можно для обновления вашего сайта, для создания специализированного сервиса и т.д.

Все запросы должны направляться к домену kodikapi.com

Все данные возвращаются в формате json, если вы не знаете, как работать с этим форматом, то скорее всего вам лучше подойдет скрипт автодобавления.

API поддерживает запросы как по http, так и по https протоколу.
Параметры можно передавать как через GET, так и через POST. Используйте те способы, которые вам больше подходят.

Ресурсы
На данный момент доступно три типа ресурсов:
1. /qualities - предназначен для получения всех возможных качеств.
2. /translations/v2, /countries, /genres, years - предназначены для получения всех возможных озвучек/стран/жанров/годов материала. Можно использовать фильтры и настройки для получения информации только для части материалов.
3. /list - предназначен для парсинга большого количество материалов (или всех материалов сразу). Можно фильтровать материалы по практически любому параметру.
4. /search - предназначен для поиска какого-либо фильма в базе. Поиск может быть как по точным параметрам фильма, так и по нечеткому названию.

/search и /list имеют огромное количество доступных для изменения параметров, так что вы точно сможете подобрать необходимый вам функционал. Если же какого-то функционала вам все равно не хватает, то можете написать об этом в поддержку и мы постараемся как можно скорее его добавить.

Описание

Для того, чтобы вам было удобнее парсить ответ API, материалы возвращаются страницами. Каждая страница содержит ограниченное количество материалов, указанное в параметре limit. После парсинга первой страницы ваш скрипт должен переходить на следующую страницу, указанную в параметре next и так далее.

Запросы отправляются по ссылке: https://kodikapi.com/list

Описание
Возвращает список наиболее подходящих вашему поисковому запросу материалов. Наиболее релевантные материалы будут идти первыми.

Результаты автоматически сортируются по последней серии и сезону. Таким образом изначально показывается озвучка с самой последней серией. Изменяя параметр приоритезации озвучек можно изменить эту логику. Также стоит учесть, что по умолчанию высокий приоритет имеют озвучки "Дублированный" и "Проф. Многоголосый" (это можно отключить настроив приоритет озвучек).

Запросы отправляются по ссылке: https://kodikapi.com/search

Документация по API плеера
Описание
Благодаря API плеера вы можете получать актуальную информацию во время воспроизведения, а также управлять плеером с помощью javascript на своем сайте. Общение с плеером происходит через функцию postMessage.

Документация разбита на два раздела:

Получение информации
Управление плеером
Получение информации
Для получения информации нужно добавить функцию обработчик входящих сообщений. message.data при этом будет содержать два параметра: key и value.

По параметру key нужно отсеивать необходимую информацию, которую вы хотите обработать. А в value будет уже сама информация. Ниже будет приведена таблица со всеми возможными значениями key.

Пример:

Копировать
<script type="text/javascript">
  function kodikMessageListener(message) {
    if (message.data.key == 'kodik_player_time_update') {
      console.log(message.data.value);
    }
  }

  if (window.addEventListener) {
    window.addEventListener('message', kodikMessageListener);
  } else {
    window.attachEvent('onmessage', kodikMessageListener);
  }
</script>

  var kodikIframe = document.getElementById("kodik-player").contentWindow;

Параметры iframe
Описание
В данной инструкции описываются параметры, добавляемые к ссылке на плеер. Сами параметры добавляются после вопросительного знака (?), но если параметров несколько, то между ними надо ставить знак амперсанд (&).

Например, если параметр один, то он указывается так: https://kodik.cc/video/93/2fc1d3b9759726b2bc5b104b225d2b4a/720p?geoblock=RU, а если мы указываем сразу несколько параметров, то это будет выглядеть так: https://kodik.cc/video/93/2fc1d3b9759726b2bc5b104b225d2b4a/720p?geoblock=RU&translations=false

Скрытие плеера для определенных стран
Отключение выбора озвучек в сериале или сезоне
Скрытие определенных озвучек в селекторе
Показ только определенных озвучек в селекторе
Автоматический выбор озвучки
Выбор серии и сезона
Показать только плеер конкретной серии или конкретного сезона в сериале
Скрытие селекторов в плеере (сезон, серия, озвучка)
Скрытие кнопки "продолжить просмотр"
Выбор времени старта
Кнопка пропуска опенинга/эндинга
Постер
Предупреждение о минимальном возрасте
Предупреждение о LGBT контенте
Скрытие плеера для определенных стран
Также вы можете глобально заблокировать все плееры на вашем сайте для какой то страны (или стран) в настройках плеера.
Для того, чтобы скрыть плеер для определенных стран, вам надо в ссылке на плеер добавить параметр geoblock, в котором перечислить через запятую коды тех стран, для которых вы хотите скрыть плеер (либо просто написать код одной страны). Коды стран можно брать из этой статьи (проматайте немного до раздела "Officially assigned code elements").

При блокировке по коду страны RU или UA плеер также будет заблокирован для Крыма.
Например, если мы имеем изначально такую ссылку https://kodik.cc/video/93/2fc1d3b9759726b2bc5b104b225d2b4a/720p и хотим заблокировать плеер для России, то ссылка будет выглядеть так: https://kodik.cc/video/93/2fc1d3b9759726b2bc5b104b225d2b4a/720p?geoblock=RU. Если мы хотим заблокировать его одновременно для России и Украины, то ссылка будет выглядеть так: https://kodik.cc/video/93/2fc1d3b9759726b2bc5b104b225d2b4a/720p?geoblock=RU,UA

Имейте в виду, что при использовании блокировщика рекламы uBlock Origin и фильтра RU Adlist (который там стоит по умолчанию) параметр geoblock стирается из ссылки плеера. При использовании блокировки по гео через настройки плеера (либо блокировки, которая проставлена в базе) такой проблемы не будет.
Отключение выбора озвучек в сериале или сезоне
Также вы можете глобально отключить выбор озвучек для всех плееров на вашем сайте в настройках плеера.
Для того, чтобы отключить выбор озвучек в плеере, вам надо в ссылке на плеер добавить параметр translations и установить его значение в false.

Например, если мы имеем изначально такую ссылку https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p и хотим отключить выбор озвучек, то ссылка будет выглядеть так: https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p?translations=false.

Скрытие определенных озвучек в селекторе
Также вы можете глобально скрыть определенные озвучки в селекторе для всех плееров на вашем сайте в настройках плеера.
Для того, чтобы скрыть определенные озвучки в плеере, вам надо в ссылке на плеер добавить параметр hide_translations и прописать в нем ID тех озвучек, которые вы хотите скрыть через запятую. ID озвучек можно посмотреть на специальной странице

Для скрытия озвучек (но не для показа только определенных озвучек), помимо ID озвучки, можно указать voice или subtitles чтобы скрыть все голосовые озвучки или все субтитры.

Например, если мы имеем изначально такую ссылку https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p и хотим скрыть озвучки 2x2 и AlexFilm, то ссылка будет выглядеть так: https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p?hide_translations=735,737.

Показ только определенных озвучек в селекторе
Также вы можете настроить это глобально для всех плееров в настройках плеера.
Для того, чтобы показывать только определенные озвучки в плеере, вам надо в ссылке на плеер добавить параметр only_translations и прописать в нем ID тех озвучек, которые вы хотите скрыть через запятую. ID озвучек можно посмотреть на специальной странице

Например, если мы имеем изначально такую ссылку https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p и хотим показывать только озвучки 2x2 и AlexFilm, то ссылка будет выглядеть так: https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p?only_translations=735,737.

Автоматический выбор озвучки
Также вы можете настроить это глобально для всех плееров в настройках плеера.
Если вы хотите, чтобы плеер сам выбирал самую верхнюю озвучку из списка (согласно включенной в глобальных настройках сортировке озвучек это может быть озвучка с самой последней серией/субтитры с самой последней серией итд), то можете включить для отдельного плеера или глобально для всех плееров автоматический выбор озвучки.

Без использования этой настройки плеер всегда будет показывать именно ту озвучку, плеер которой вы вставили. И уже внутри этой озвучки будет показываться настроенная вами серия (последняя в последнем сезоне/первая в первом итд). То есть без этой настройки, даже если есть другая озвучка с более актуальной последней серией, она не будет выбрана.

Для включения, необходимо добавить в ссылку плеера параметр auto_translation=true.

Например, если мы имеем изначально такую ссылку https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p и хотим включить для неё автоматический выбор озвучки, то ссылка будет выглядеть так: https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p?auto_translation=true.

Выбор серии и сезона
Также вы можете глобально настроить выбираемую серию и сезон для всех плееров на вашем сайте в настройках плеера.
С помощью данной настройки можно выбрать какая серия и сезон будут показываться новому пользователю в плеере. Важно понимать, что как только пользователь выберет серию и сезон самостоятельно, то для данного сериала его выбор сохранится и данные настройки будут игнорироваться.

Настройка через параметр iframe имеет превосходство над глобальной настройкой в настройках плеера, поэтому вы можете использовать и то и то. Например выбрать желаемую серию и сезон глобально, но для каких-то определенных сериалов указывать данный параметр вручную через параметры iframe.

Для выбора серии и/или сезона необходимо передать плееру season и/или episode параметр, содержащий числовое значение без пробелов. Не переживайте, если вы случайно укажете серию или сезон, которой в сериале нет, то параметр просто не будет учитываться.

Например, если мы имеем изначально такую ссылку https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p и хотим выбрать сезон и серию, то ссылка будет выглядеть так: https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p?season=2&episode=3.

Показать только плеер конкретной серии или конкретного сезона в сериале
Каждая серия или сезон имеет свою собственную ссылку и свой собственный плеер, которые можно взять либо кликнув по кнопке "сезонов" в базе, либо нажав кнопку "сезон" в нашем модуле, либо через API, если вы разработчик.
Для тех случаев, когда вы не можете взять сразу прямую ссылку на сезон или серию, но у вас есть ссылка на сериал, есть два параметра: only_episode и only_season.

Достаточно указать к ссылке на сериал сезон через season параметр и параметр only_season поставить в true и вместо плеера сериала у вас будет показываться плеер сезона, в нем не будет выбора сезонов, но будет выбор серий и озвучек. Если вы хотите убрать еще и выбор озвучек, то сделать это можно с помощью параметра translations=false, как описано в инструкции выше.

Аналогично к ссылке на сериал можно указать параметры episode и season и добавить параметр only_episode=true. В таком случае вместо плеера сериала у вас будет показывать плеер конкретного эпизода. В нем не будет ни выбора сезонов, ни выбора серий, ни выбора озвучек.

Вы можете не указывать параметр season при использовании only_season и параметры season и episode при использовании only_episode. В таком случае выбранный сезон или выбранная серия будут соответствовать тому сезону или серии, который был бы показан в плеере без использования этих параметров. То есть все остальные iframe параметры и настройки сайта также будут влиять на выбор этого сезона или серии.
Например, если мы имеем изначально такую ссылку https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p и хотим показать конкретную серию, то ссылка будет выглядеть так: https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p?season=2&episode=3
&only_episode=true.

Если же мы хотим показать конкретный сезон, то ссылка будет выглядеть так: https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p?season=2&only_season=true

Скрытие селекторов в плеере (сезон, серия, озвучка)
От параметра only_episode этот параметр отличается тем, что не отключает функционал выбора серии и сезона, а только скрывает видимость этого интерфейса для пользователя (сам плеер остается плеером сериала/сезона, а не плеером серии). Но использовать этот функционал все еще можно через API плеера.

Так как с данным параметром плеер остается плеером сериала/сезона, то в нем будет работать автопереключение серий, сохранение последней просмотренной серии (кнопка "продолжить просмотр") и все, что связано с плеером сериалов.

Более детально настроить работу плеера сериала вы можете в настройках своего сайта.
Чтобы скрыть селекторы серий, сезонов и озвучек, вам надо в ссылке на плеер добавить параметр hide_selectors и установить его значение в true.

Например, если мы имеем изначально такую ссылку https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p и хотим скрыть селекторы, то ссылка будет выглядеть так: https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p?hide_selectors=true.

Скрытие кнопки "продолжить просмотр"
Чтобы скрыть кнопку "продолжить просмотр" для конкретного плеера вам надо в ссылке на плеер добавить параметр hide_resume_button и установить его значение в true.

Например, если мы имеем изначально такую ссылку https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p и хотим скрыть кнопку "продолжить просмотр", то ссылка будет выглядеть так: https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p?hide_resume_button=true.

Выбор времени старта
С помощью данной настройки можно выбрать с какого времени плеер будет стартовать. Обычно мы сами устанавливаем подобный параметр для фильмов или сериалов, в которых необходимо пропустить встроенную рекламу, но если по какой-то причине вы хотите выбрать это время самостоятельно, то у вас есть такая возможность.

Для выбора времени стартра вам необходимо передать плееру start_from параметр, содержащий числовое значение без пробелов. Время указывается в секундах.

Для сериалов данный параметр будет влиять только на изначально выбранную серию. В случае выбора пользователем другой серии (или если другая серия выберется автоматически, так как она сохранилась у пользователя при предыдущем выборе) параметр сбрасывается.

Например, если мы имеем изначально такую ссылку https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p и хотим выбрать время старта, то ссылка будет выглядеть так: https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p?start_from=300.

Кнопка пропуска опенинга/эндинга
Данный параметр позволяет добавить в плеер кнопку пропуска определенного интервала (например опенинга или эндинга).

Для настройки необходимо передать в параметр skip_button интервалы, для которых вы хотите добавить пропуск.

Начало и конец интервала разделяются дефисом (-).
Сами интервалы между собой разделяются запятой (,).
Время начала или конца интервала может указываться в секундах (29), минутах с секундами (12:29), часах с минутами и секундами (3:12:29).
Примеры интервалов: 12-30, 12:20-30:10, 40-1:10:12

У интервалов также есть настраиваемые названия. Они влияют на надпись на кнопке пропуска и на подсказку над интервалом на таймлайне плеера. На данный момент доступны 4:

opening (опенинг)
ending (эндинг)
intro (вступление)
credits (титры)
intro и credits названия ставятся автоматически. Если вы хотите настроить название вручную, то его нужно указать в квадратных скобках ([]) перед интервалом. Например: [opening]12-60

Например, если мы имеем изначально такую ссылку https://kodik.cc/seria/5946/3855abb1af58a08bf6570822e0a6a751/720p и хотим добавить два интервала для опенинга и эндинга, то ссылка будет выглядеть так: https://kodik.cc/seria/5946/3855abb1af58a08bf6570822e0a6a751/720p
?skip_button=[opening]12-60,[ending]24:12-26:15.

Постер
Если вам нужно установить одинаковый постер на все плееры, то сделать это лучше через настройки плеера в соответствующем разделе.
С помощью данной настройки можно настроить уникальный постер для одного конкретного плеера, который будет показан до запуска плеера.

Для добавления постера вам необходимо передать плееру poster параметр, содержащий прямую ссылку на изображение.

Лучше всего загружать изображения на свой сервер и вставлять ссылки уже оттуда, но если у вас нет такой возможности или это сложно для вас, то можете воспользоваться следующими сервисами, которые предоставляют прямые ссылки на изображения после загрузки: vgy.me, imgsafe.org, imgbox.com, cubeupload.com.

Например, если мы имеем изначально такую ссылку https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p и хотим добавить в нее свой постер, то ссылка будет выглядеть так:: https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p
?poster=https://i.vgy.me/O1GHuf.jpg.

Предупреждение о минимальном возрасте
Вы можете глобально включить для всех плееров иконки в превью/плеере или подтверждение возраста для материалов с 18+ рейтингом в настройках плеера.
По умолчанию мы автоматически парсим максимально строгие ограничения из Shikimori или КиноПоиска. Но если вам по какой-то причине нужно изменить рейтинг для конкретного плеера или включить/отключить подтверждение возраста только для одного плеера, то сделать это вы можете с помощью следующих параметров.

min_age - Минимальный возраст. Передаете просто число, например min_age=18.

min_age_confirmation - подтверждение возраста (если минимальный возраст 18). Ставите либо true, либо false. Например: min_age_confirmation=true. Если подтверждение включено, но минимальный возраст меньше 18, то оно не будет показано.

Например, если мы имеем изначально такую ссылку https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p и хотим изменить возраст на 18 и включить подтверждение, то ссылка будет выглядеть так: https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p?min_age=18
&min_age_confirmation=true.

Предупреждение о LGBT контенте
Вы можете глобально включить для всех плееров иконки в превью или предупреждение для материалов с LGBT контентом в настройках плеера.
Мы самостоятельно проставляем к материалам LGBT метки, согласно которым в плеере будет показана иконка или предупреждение (если вы включите эту функцию). Но если вам по какой-то причине нужно включить/отключить иконку/предупреждение только для одного плеера, то сделать это вы можете с помощью следующих параметров.

lgbt_preview_icon - Иконка LGBT в плеере. Значения могут быть true, либо false. Например lgbt_preview_icon=true. Если вы передали этот параметр, то иконка будет показана в любом случае. Вне зависимости от того, помечен ли в базе этот материал, как содержащий LGBT контент.

lgbt_confirmation - Предупреждение о LGBT контенте. Значения могут быть true, либо false. Например: lgbt_confirmation=true. Если вы передали этот параметр, то предупреждение будет показано в любом случае. Вне зависимости от того, помечен ли в базе этот материал, как содержащий LGBT контент.

Например, если мы имеем изначально такую ссылку https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p и хотим включить иконку и предупреждение о LGBT контенте для данного плеера, то ссылка будет выглядеть так: https://kodik.cc/serial/5946/3855abb1af58a08bf6570822e0a6a751/720p?lgbt_preview_icon=true
&lgbt_confirmation=true.
